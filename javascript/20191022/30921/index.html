JavaScript - ES6 - 樣板字串

發表於 2019-10-22 分類於 JavaScript

以往在組字串時，都是透過 引號 '、" 和 加號 + 來組合，換行也必須在結尾加上 反斜線 \，撰寫上容易混亂，閱讀也不夠自然，因此 ES6 新增了組字串的新方法，組起來也較為接近原生的 HTML

ES6 字串基本寫法

ES6 新增的方法如下

  • 使用 重音符 ` 來取代引號包住字串

  • 使用 ${ } 來包住 “變數” 或 “函式”，取代加號 + 來連接

    • 如果需要插入重音符 ` 或其他特殊字元，可以使用 “反斜線” \ 來進行轉義
